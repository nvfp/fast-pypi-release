## Changelog file name format

Pattern: `.*(?i)changelog.*`

Explanation: The name can be anything, but it should at least include the word 'changelog' (not case-sensitive).

Examples ✔️:
- changelog
- CHANGELOG
- CHANGELOG.md
- CHANGELOG.txt
- Changelog.txt
- 123 - Changelog.md
- 123 - Changelog - Foo.md
